# 线性筛法
```cpp
void GetPrime(int N) {
    for (int i = 2; i <= N; ++i) {
        if (!vis[i]) {
            prime[len++] = i;
        }
        for (int j = 1; j < len && i * prime[j] <= N; ++j) {
            vis[i * prime[j]] = 1; // 划掉可能的合数
            if (i % prime[j] == 0) {
             // 如果 i 本身就是个素数，那么枚举到自己时停止
             // 如果是合数，那么枚举到它最小的素因子停止
                break;
            }
        }
    }
}
```
